## Estructura del proyecto
- `index.html` - Página principal con 3 secciones
- `views/productos.html` - Galería de productos en cuadrícula
- `views/presupuesto.html` - Formulario de presupuesto con validación
- `views/contacto.html` - Página de contacto con mapa
- `css/style.css` - Hoja de estilos principal
- `img/` - Imágenes y recursos

## Características implementadas
- Diseño responsive (mobile-first)
- Formularios con validación HTML5
- Navegación con página activa resaltada
- Galería interactiva con modal
- Mapa de Google Maps integrado
- Footer con redes sociales
- Código HTML validado por W3C Validator
